Projects

Planning — Autonomous Racing Robot

C++ · ROS · PCL · RANSAC

ROS-based autonomous racing robot with Point Cloud Library for real-time LiDAR processing. Implements RANSAC-based line detection and path planning algorithms for three competitive racing events.

🏎️ Acceleration 加速赛

Straight-line speed run using RANSAC line detection on LiDAR point clouds. The robot detects track boundaries and follows the optimal center path at maximum velocity.

🔄 Skidpad 绕桩赛

Figure-8 navigation using skidpad path publisher and cone detection. The robot calculates optimal turning radius through two circular laps while maintaining stability.

🏁 Tracking 追踪赛

Multi-lap boundary detection with lap counting. Uses PCL to process LiDAR data, detect track edges, and navigate the closed-loop circuit at competitive speeds.
