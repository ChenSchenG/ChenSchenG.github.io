Projects

FSAC Planning — 无人方程式规划模块

C++ · ROS · PCL · CGAL · Eigen

中国大学生无人驾驶方程式大赛（FSAC）路径规划模块。基于 ROS 与 PCL 处理 LiDAR 点云数据，为三个竞速赛项实现了不同的感知与规划算法，完成从点云到路径的全链路闭环。

🏎️ Acceleration 加速赛

基于 RANSAC 随机采样一致性对 LiDAR 点云进行左右边界线拟合（Eigen QR 分解），取中线作为行驶路径。支持双车道中线融合与单车道偏移回退，EMA 时序平滑（α=0.3）消除抖动。

🔄 Skidpad 八字绕环

两阶段点云配准定位：先通过 SAC-IA（FPFH 特征 + 采样一致性初始对齐）获得粗匹配，再经 ICP（迭代最近点）精配准，输出 3-DOF 平面变换矩阵（x, y, yaw），实现与先验地图的实时对齐。

🏁 Trackdrive 循迹赛

使用 CGAL Delaunay 三角剖分连接左右锥桶点云，筛选跨类型边并提取中点作为候选路径点，再通过贪心排序（距离 + 航向角 + 前方奖励加权评分）生成有序路径。首圈保守截断，后续圈全图扩展。

Python · Rich TUI · LLM · Security
